# VWD温馨家园 - 项目说明

## 项目概述

VWD温馨家园是一个为血管性血友病（Von Willebrand Disease, VWD）患者及家属提供的公益科普网站。本网站提供权威、易懂的VWD疾病知识、治疗信息、护理指南和心理支持，致力于打造一个可信赖的线上家园。

## 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式设计（响应式设计）
- **原生JavaScript** - 交互功能
- 无需框架或构建工具，可直接部署至任何静态网站托管服务

## 项目结构

```
vwd-home/
├── index.html          # 首页
├── knowledge.html      # 疾病知识页
├── treatment.html      # 治疗与生活页
├── support.html        # 患者支持页
├── about.html          # 关于我们页
├── css/
│   └── style.css       # 全局样式文件
├── js/
│   └── main.js         # JavaScript交互文件
├── images/
│   └── README.md       # 图片资源说明
└── README.md           # 本文件
```

## 网站特点

### 设计特点
- **配色方案**：蓝色系主色调，搭配浅蓝背景和深灰文字，绿色点缀
- **响应式设计**：完美适配手机、平板和电脑等所有设备
- **视觉风格**：参考"血友之家"，做到清晰、专业、简洁、温馨
- **统一性**：所有页面共享一致的导航栏和页脚

### 功能特点
1. **响应式导航栏**：移动端自动折叠为汉堡菜单
2. **FAQ折叠面板**：支持展开/收起交互
3. **侧边栏导航**：支持锚点定位和高亮
4. **平滑滚动**：点击锚点链接平滑滚动到目标位置
5. **图片占位**：使用Unsplash免费可商用图片

### 内容模块
- **首页**：英雄横幅、快速入口、最新动态、重要提示
- **疾病知识**：VWD定义、分型、症状与诊断、遗传模式
- **治疗与生活**：治疗目标、常用药物、出血处理、手术准备、日常生活
- **患者支持**：心理支持、患者故事、FAQ、资源链接
- **关于我们**：网站宗旨、使命、愿景、免责声明

## 快速开始

### 本地预览

1. 直接双击 `index.html` 文件，在浏览器中打开
2. 或使用本地服务器：
   ```bash
   # 使用Python启动简单服务器
   python -m http.server 8000
   
   # 然后访问 http://localhost:8000
   ```

### 部署到GitHub Pages

1. 将项目推送到GitHub仓库
2. 进入仓库设置页面
3. 找到"Pages"设置
4. 选择主分支作为源
5. 保存后，网站将自动部署

### 部署到其他静态托管服务

本网站可以部署到任何静态网站托管服务，例如：
- Netlify
- Vercel
- Cloudflare Pages
- 阿里云OSS
- 腾讯云COS

## 自定义说明

### 替换图片

参考 `images/README.md` 文件，下载相应的图片并放置到 `images/` 目录，然后修改HTML中的 `src` 属性。

### 修改配色

编辑 `css/style.css` 文件，在 `:root` 部分修改CSS变量：

```css
:root {
  --primary-color: #4a90e2;      /* 主色调 */
  --primary-dark: #357abd;       /* 主色调深色 */
  --primary-light: #e3f2fd;      /* 主色调浅色 */
  /* ... 其他变量 */
}
```

### 添加内容

直接编辑对应的HTML文件，按照现有的HTML结构添加新的内容块。
